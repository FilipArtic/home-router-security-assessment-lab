## Scope

The assessment was performed on a consumer'grade GPON router Within a controlled local network enviroment.

Router Security Assessment

Dispositivo: ZTE ZXHN F670L1FXS

1. Información General

IP: 192.168.x.x
Hostname: internal-gateway
Fabricante: ZTE
Modelo: ZXHN F670L1FXS
Tipo: Router GPON
Sistema Operativo: Linux (Kernel 3.x – 4.x)
Entorno: Laboratorio controlado

2. Metodología

Se realizó un análisis de seguridad en un entorno controlado utilizando:

Escaneo de red con Nmap
Inspección de servicios expuestos
Análisis de interfaz web
Evaluación de autenticación
Observación de comportamiento del sistema

10. Nivel de Riesgo Global

MEDIO

Buenas prácticas implementadas

11. Recomendaciones

Deshabilitar Telnet completamente
Restringir acceso al DNS
Deshabilitar HTTP si no es necesario
Cambiar credenciales por defecto
Evitar exposición de etiqueta física
Auditar endpoints internos
Validar controles del lado servidor
Deshabilitar WPS

12. Conclusión

El dispositivo presenta una configuración de seguridad aceptable para entornos domésticos, incorporando mecanismos básicos de protección como:

HTTPS
Control de sesiones
Protección contra fuerza bruta

Sin embargo, mantiene una superficie de ataque moderada, principalmente debido a:

Servicios expuestos
Dependencia de credenciales físicas
Amplia funcionalidad administrativa
